## 공개 입력 데이터

| 파일 | 역할 | 게임 원문 포함 여부 |
|---|---|---|
| `translations/hyda_ko.json` | 한국어 화자·본문 434개 및 653개 적용 위치 | 일본어 원문 없음 |
| `translations/hyda_patch_manifest.json` | 해시·오프셋 기반 compact 구조 매니페스트 | 원문·raw bytes 없음 |

compact 매니페스트는 이벤트 뱅크별 archive/stream 지문, 세그먼트 SHA-256,
화자 모드, 메시지 경계 오프셋과 원문 줄 수만 제공합니다. 추출 일본어 텍스트,
메시지 raw bytes, bitmap과 기타 게임 리소스는 배포하지 않습니다.

## 릴리스 ZIP 점검

릴리스 ZIP에는 xdelta 패치, 한국어 사용법, `SHA256SUMS.txt`, NanumSquare Neo
저작권 고지와 OFL 1.1 전문만 포함합니다. 다음 항목이 들어 있으면 배포하지
않습니다.

- 원본 또는 패치된 ISO
- 게임 실행 파일과 추출 게임 데이터
- 일본어 원문 전체나 원문 message raw bytes
- BIOS, 에뮬레이터, 세이브스테이트
- `NanumSquareNeo-cBd.ttf` 파일 자체

최종 ZIP을 별도 디렉터리에 풀어 정확히 다음 5개 파일만 있는지 확인했고,
`SHA256SUMS.txt`의 네 항목을 모두 다시 계산해 일치함을 확인했습니다.

- `SO3_DC_Disc1_Korean_Hyda_653_Nanum_v0.4.0-alpha.1.xdelta`

- `SHA256SUMS.txt`
- `NanumSquareNeo-COPYRIGHT.txt`

## 런타임 상태

이 릴리스 준비 과정에서는 에뮬레이터를 실행하지 않았습니다. 이는 사용자의 요청에
따른 의도적인 범위 제한입니다. 따라서 다음 항목은 사용자 런타임 검증 대상으로
남습니다.

- 653개 대상 대사가 실제 진행 순서와 문맥에서 올바르게 표시되는지
- 화자 이름, 줄바꿈, 대기와 진행 흐름에 회귀가 없는지
- 24×24 NanumSquare Neo 글리프가 화면에서 깨지거나 잘리지 않는지
- 번역 문장이 대사창 폭을 넘거나 읽기 어렵지 않은지
- 하이다 구간의 이벤트와 게임 진행이 정상적으로 이어지는지

정적 검증 통과와 xdelta 해시 일치는 실제 에뮬레이터 출력 확인을 대신하지
않습니다.
